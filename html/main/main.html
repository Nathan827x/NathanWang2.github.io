Restaurant Roulette

From July 2017

To Present

Description:

This application is for those who are always wondering what to eat. The application will select a random restaurant to go to based on a user specified location.

React was used to create the front end forms and the page layout. It integrates Google Maps APIs to show a map on the main page. Once you click submit, the map will then change to the location of the random restaurant by an API call to the server.

The back end server was written in Python using the Flask micro framework. This gave me experience with creating REST APIs with Python. Not only did I create my own REST APIs, I also was able to use Google Places and Google Details API to gather more information. Once the data was retrieved it is stripped to ensure that only the essential information is sent to the front end to display.

If you would like to keep up with the project look on my GitHub.

Tech Stack:

  • React (Frontend)
  • Flask (Python Microframework)(Backend)
  • SQLAlchemy (Database)

Objectives:

  • To get experience in creating a full stack web application
  • Experience with React and integrating APIs from a 3rd party
  • Create and Implement REST API using Python with Flask
  • Learn to use 3rd party APIs to gather information and read documentation using Python in the backend
  • Experience in deploying an application
